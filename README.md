## ­ЪЏа№ИЈ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃў рЃЊрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃў

### **рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў**
- **.NET 8** - рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў Microsoft-рЃўрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЎрЃў
- **Entity Framework Core** - ORM рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЎрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА
- **SQL Server** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

### **рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў рЃЊрЃљ рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃў**
- **Clean Architecture** - рЃарЃЮрЃЉрЃћрЃарЃб рЃЏрЃљрЃарЃбрЃўрЃюрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў
- **CQRS (Command Query Responsibility Segregation)** - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў
- **SOLID рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў** - рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃќрЃћ рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў
- **Repository Pattern** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ
- **Unit of Work Pattern** - рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

### **рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃў**
- **Cursor AI** - AI-powered рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў рЃЊрЃљ рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў

### **рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ**
> **рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў**: рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃў (Cursor AI) рЃўрЃДрЃЮ рЃЕрЃћрЃЏрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃњрЃћрЃњрЃЏрЃўрЃА рЃарЃћрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃгрЃДрЃљрЃарЃЮ, рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ, design pattern-рЃћрЃЉрЃў рЃЊрЃљ SOLID рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃўрЃА рЃЊрЃљрЃфрЃЋрЃљ рЃАрЃарЃБрЃџрЃљрЃЊ рЃЏрЃљрЃЦрЃЋрЃА рЃюрЃљрЃцрЃўрЃЦрЃарЃў рЃЏрЃћ рЃЊрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ рЃЕрЃћрЃЏрЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃўрЃЌ рЃљрЃарЃўрЃА рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў. AI рЃўрЃДрЃЮ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃћрЃарЃўрЃАрЃљрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЕрЃћрЃЏрЃў рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃљ.
