# Interview Project - рЃърЃћрЃарЃАрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

## ­ЪЊІ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

рЃћрЃА рЃљрЃарЃўрЃА **рЃърЃћрЃарЃАрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ Clean Architecture рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃърЃўрЃарЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ, рЃгрЃљрЃерЃџрЃљ рЃЊрЃљ рЃФрЃћрЃЉрЃюрЃљ.

## ­ЪЈЌ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ **Clean Architecture** рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃцрЃћрЃюрЃћрЃЉрЃўрЃАрЃњрЃљрЃю:

```
Interview/
РћюРћђРћђ Interview.Api/              # API Layer (Controllers, Middleware)
РћюРћђРћђ Interview.Application/      # Application Layer (Commands, Queries, Handlers)
РћюРћђРћђ Interview.Domain/           # Domain Layer (Entities, Value Objects)
РћюРћђРћђ Interview.Infrastructure/   # Infrastructure Layer (Database, Repositories)
РћћРћђРћђ Interview.Tests/            # Test Projects
    РћюРћђРћђ Interview.Tests.Unit/   # Unit Tests
    РћћРћђРћђ Interview.Tests.Integration/ # Integration Tests
```

рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃљрЃдрЃгрЃћрЃарЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃўрЃ«рЃўрЃџрЃЮрЃЌ ArchitectureReadme.md рЃцрЃљрЃўрЃџрЃерЃў

### ­ЪЊЂ рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

#### **Interview.Api** - API рЃцрЃћрЃюрЃљ
- **Controllers**: HTTP endpoints
- **Middleware**: Global exception handling, Localization
- **Extensions**: Service configuration

#### **Interview.Application** - рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃцрЃћрЃюрЃљ
- **Commands**: CQRS Commands (Create, Update, Delete)
- **Queries**: CQRS Queries (Get, Search, Reports)
- **Handlers**: Command/Query handlers
- **DTOs**: Data Transfer Objects

#### **Interview.Domain** - рЃЊрЃЮрЃЏрЃћрЃюрЃўрЃА рЃцрЃћрЃюрЃљ
- **Entities**: Person, PersonPhoneNumber, PersonRelation
- **Value Objects**: Name, PersonalNumber, BirthDate, Age
- **Dictionaries**: City, Gender, PhoneType, RelationType

#### **Interview.Infrastructure** - рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃцрЃћрЃюрЃљ
- **Data**: Entity Framework DbContext, Migrations
- **Repositories**: Data access implementations
- **Services**: File storage, Localization

## ­Ъїљ API Endpoints

- `GET /api/persons` - рЃДрЃЋрЃћрЃџрЃљ рЃърЃћрЃарЃАрЃЮрЃюрЃўрЃА рЃАрЃўрЃљ
